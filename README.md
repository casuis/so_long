# SO_LONG

_Voici mon projet So_long pour l'ecole 42_

## Idée Global

Création d'un jeu 2D à l'aide de l'utlisation de la MLX

## Compilation

```
$ make
```

## Execution

Il y a plusieurs maps à dispositions dans le dossier 'maps/'
```
$ ./so_long map/map_correct.ber
```
